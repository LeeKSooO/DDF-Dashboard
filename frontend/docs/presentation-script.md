# DDF-MSTGCN 대시보드 발표 대본

## 🎯 1. 시작 (30초)

안녕하세요. 저는 프론트엔드 담당자 박상훈입니다.

오늘 발표할 주제는 **서울시 수요응답형 교통서비스(DRT) 도입을 위한 지능형 분석 대시보드**입니다.

현재 서울시는 대중교통 사각지대 해소와 효율적인 교통서비스 제공을 위해 DRT 서비스 도입을 검토하고 있습니다. 하지만 어디에, 언제, 어떤 방식으로 DRT를 도입할지에 대한 데이터 기반 의사결정 도구가 부족한 상황입니다.

저희는 MSTGCN 모델과 실제 교통 데이터를 활용하여 DRT 서비스 최적 도입 지점을 분석하고 시각화하는 통합 대시보드를 개발했습니다.

---

## 📊 2. 시스템 개요 (1분)

### 2.1 전체 대시보드 구조

저희 시스템은 **5개의 주요 분석 탭**으로 구성되어 있습니다.

1. **대시보드 개요** - 전체 DRT 분석 현황을 한눈에 파악
2. **히트맵** - 지역별 수요 패턴을 색상으로 직관적 시각화
3. **DRT 분석** - 정류장별 DRT 적합성을 상세 분석
4. **교통 분석** - 이상 패턴 및 특화 구간 식별
5. **OD 분석** - 우선순위별 Origin-Destination 플로우 분석

### 2.2 핵심 설계 철학

- **데이터 기반 의사결정**: 실제 버스 이용 데이터와 AI 모델 예측 결합
- **직관적 시각화**: 복잡한 교통 데이터를 이해하기 쉬운 차트와 지도로 표현
- **실무진 친화적 UI**: 교통 정책 담당자가 쉽게 사용할 수 있는 인터페이스
- **실시간 분석**: 월별, 지역별 필터링을 통한 유연한 데이터 탐색

## 🚀 3. 핵심 기능 시연 (3분)

### 3.1 DRT 분석 탭 (1분 20초)

**[DRT 분석 탭 시연]**

DRT 분석 탭은 수요응답형 교통서비스 도입의 핵심 분석 화면입니다.

**좌측 지도 영역**에서는:
- 서울시 전체 정류장별 DRT 적합도 점수를 색상으로 표시
- 구를 클릭하면 해당 지역 정류장들이 나타남
- 개별 정류장 클릭 시 상세 분석 정보 제공

**우측 상단**에는 선택된 정류장의 **24시간 DRT 점수 변화**를 라인 차트로 표시:
- 시간대별 수요 패턴 파악 가능
- 최적 운행 시간대 식별
- 일평균 대비 변화량 확인

**우측 하단**에는 **정류장 피크 특성 분석**:
- 교통취약지형: 접근성, 사회적 형평성 지표
- 출퇴근형: 시간 집중도, 피크 수요 비율
- 관광형: 관광 집중도, 특정 시간대 밀도

이를 통해 각 정류장에 가장 적합한 DRT 모델을 제안할 수 있습니다.

### 3.2 교통 분석 탭 (1분 20초)

**[교통 분석 탭 시연]**

교통 분석 탭에서는 **이상 패턴 및 특화 구간**을 분석합니다.

**지역 특성 분석**:
- 주거지역 vs 업무지역 정류장 분류
- 각 지역 특성에 맞는 DRT 서비스 유형 제안

**저활용 정류장**:
- 수요 대비 서비스 공급이 부족한 지역 식별
- DRT 서비스 우선 도입 대상 선정

**시간대별 특화 정류장**:
- 🌅 **러시아워 고수요**: 오전 6-8시, 오후 5-7시 집중 수요 지역
- 🎯 **주말 우세**: 관광지, 쇼핑몰 인근 레저 수요 집중
- 🌙 **심야 고수요**: 유흥가, 24시간 시설 주변
- 🍽️ **점심시간 특화**: 업무지구 맛집가, 직장인 외식 수요

각 카드는 실시간 데이터 기반 애니메이션과 함께 순위별 정류장을 표시하여 시각적 효과를 극대화했습니다.

### 3.3 OD 분석 탭 (20초)

**[OD 분석 탭 시연]**

가장 혁신적인 OD 분석 탭입니다.

**3D 시각화**를 통해 Origin-Destination 플로우를 입체적으로 표현:
- **P1 고수요/저수요 환승구간**: 빨간색/주황색으로 긴급도 표시
- **P2 직행노선 부족**: 파란색으로 개선 필요 지역 표시  
- **P3 장거리 저수요**: 보라색으로 통합 검토 대상 표시

정류장 클릭 시 해당 지점의 모든 OD 연결을 실시간으로 시각화하며, 우측 패널에서 상세 통계를 제공합니다.

우클릭 비활성화, 컴팩트한 필터 박스 등 사용자 경험을 고려한 세부 기능들도 구현되어 있습니다.

## 💻 4. 기술적 성과 (1분)

### 4.1 AI 모델 통합
- **MSTGCN(Multi-Scale Temporal Graph Convolutional Network)** 모델을 프론트엔드에 완전 통합
- 3개 특화 모델(교통취약지형, 출퇴근형, 관광형) 실시간 전환
- 월별 97.2% 정확도의 DRT 적합도 예측

### 4.2 고성능 데이터 처리
- **Next.js 14** 기반 서버사이드 렌더링으로 초기 로딩 속도 최적화
- **React Query**를 활용한 효율적인 API 캐싱 및 상태 관리
- **Abort Controller** 패턴으로 불필요한 API 요청 취소 및 메모리 누수 방지

### 4.3 시각화 기술
- **DeckGL**을 활용한 WebGL 기반 3D 지도 렌더링
- **Recharts** 기반 반응형 차트로 모든 디바이스 대응
- **Tailwind CSS**와 **Framer Motion**으로 현대적 UI/UX 구현

### 4.4 사용자 경험 최적화
- 로딩 상태, 에러 처리, 스켈레톤 UI 등 완성도 높은 인터액션
- 애니메이션 기반 숫자 카운트업으로 시각적 몰입도 향상
- 반응형 디자인으로 데스크탑/태블릿 모든 환경 지원
